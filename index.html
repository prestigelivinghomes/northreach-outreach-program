📋 How to set this up in Make.com (5 minutes)

  1. Create a new scenario in Make.com — go to make.com → Create a new scenario
  2. Add a Webhooks trigger — search for "Webhooks" → choose Custom webhook → click Add → name it "North Reach Outreach" → copy the webhook URL it gives you
  3. Paste the URL above — paste your Make.com webhook URL into the field above and click Test to confirm it's working
  4. Add a GoHighLevel module — click the + after the webhook → search "GoHighLevel" → choose Create/Update Contact → connect your GHL account
  5. Map the fields — Make.com will show you the data fields from North Reach. Map them like this:
    • first_name → First Name
    • last_name → Last Name
    • email → Email
    • phone → Phone
    • company_name → Company Name
    • suburb → Address (City)
    • tags → Tags (set to "outreach-lead")
    • source → Lead Source (set to "North Reach Outreach")
  6. Turn on the scenario — click the toggle at the bottom left to activate it. From now on, every time you mark a prospect as Replied or Won, they appear in GHL automatically.
